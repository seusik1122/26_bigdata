# Git 기본 명령어

## 현재 위치 확인
```powershell
pwd
```
![pwd](스크린샷%202026-03-20%20104523.png)

---

## 파일 목록 확인
```powershell
ls
```
![ls](스크린샷%202026-03-20%20104608.png)

---

## 폴더 이동
```powershell
cd ch2
```
![cd](스크린샷%202026-03-20%20104647.png)

---

## 상위 폴더로 이동
```powershell
cd..
```
![cd..](스크린샷%202026-03-20%20104719.png)

---

## 폴더 생성
```powershell
mkdir ch4
```
![mkdir](스크린샷%202026-03-20%20104748.png)

---

## 파일 내용 확인
```powershell
cat README.md
```
![cat](스크린샷%202026-03-20%20104953.png)

---

## 파일 복사
```powershell
cp README.md ch2\README.md
```
![cp](스크린샷%202026-03-20%20105121.png)

---

## 파일 이동
```powershell
mv README.md ch4
```
![mv](스크린샷%202026-03-20%20105238.png)

---

## 파일 삭제
```powershell
rm ch2\README.md
```
![rm](스크린샷%202026-03-20%20105314.png)

---

## 폴더 삭제
```powershell
rm -Recurse ch4
```
![rm -Recurse](스크린샷%202026-03-20%20105358.png)

---

## 화면 지우기
```powershell
clear
```
![clear](스크린샷%202026-03-20%20105430.png)
